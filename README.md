# 考勤统计汇总

一个很简单的本地考勤统计工具，直接读取当前目录里的考勤机导出文件。

## 当前规则

- 重点读取 `考勤记录` sheet。
- 普通人员固定扣除 `12:00-13:00` 午休。
- `任杰`、`王清玲`、`梁宗茂` 默认按 `12:00-14:00` 午休；如果 `12:00-14:00` 之间有打卡，按最早午间打卡恢复计时。
- 一天首个 `06:00` 前打卡，且当天后面还有白天打卡时，会自动并回前一天，按跨天加班处理。
- 工作时按整小时累计，余数每满 `20` 分钟记 `0.5`，不足 `20` 分钟不计。

## 使用方法

1. 编译启动

```bash
mvn spring-boot:run
```

2. 浏览器打开

`http://127.0.0.1:8080/app/index.html`

页面里会展示:

- 每人整月汇总
- 每人每天明细
- CSV / Excel 导出
